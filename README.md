jaj_taglib
==========

Just another JSP tag library

How to make the JAR file
------------------------
1. create the *${project_root}/lib/* directory
2. copy the JSP API jar file into *${project_root}/lib/* (the compilation was tested with JSP API 2.0)
3. run ant within the *${project_root}* directory
4. pick *jaj_taglib.jar* from *${project_root}/build/*
